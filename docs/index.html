What you need

  • 01
    Python 3.10+
    Check: python3 --version
    Install (macOS): brew install python3
    Install (Windows): python.org/downloads
  • 02
    Claude — CLI or desktop app
    CLI: npm install -g @anthropic-ai/claude-code
    Desktop: claude.ai/download (for Cowork launcher)
  • 03
    cloudflared
    macOS: brew install cloudflare/cloudflare/cloudflared  ·  Windows: winget install Cloudflare.cloudflared
  • 04
    A Google account
    For Drive API access
  • 05
    A Plaud device with Google Drive sync on
    Transcripts must auto-sync to a Drive folder

Installation

1

Clone the repo

Copy$ git clone https://github.com/OllieB1943/plaude.git
$ cd plaude
2

Install Python deps

Copy$ pip install -r requirements.txt
3

Run the setup wizard

Handles Google OAuth, folder selection, and config in one shot.

Copy$ python setup.py
4

Enable Plaud → Drive sync

In the Plaud app, turn on Google Drive sync and point it to the same folder you configured above. Check the Plaud help docs for where the setting lives.

5

Start the bridge

Copy$ ./start.sh

How it works

🎙
Record on Plaud Speak naturally — tasks, ideas, anything
→
💬
Say "send to claude" Trigger phrase detected automatically
→
⚡️
Claude takes over Via CLI or Cowork — your choice

Choose your launcher

Set "launcher" in config.json to control how Claude is triggered.

💻
CLI default
Writes a CLAUDE.md into the matched project directory and launches Claude Code in a terminal.
"launcher": "cli"
🤖
Cowork
Drops a task file into an inbox folder that Claude Cowork watches. Works for anything — code, writing, planning, emails.
"launcher": "cowork"
⚡️
Both
Triggers CLI and Cowork simultaneously. Set "launcher": "both" in config.
Setting up Cowork
  1. Set "launcher": "cowork" and "cowork_inbox": "~/claude-inbox" in config.json
  2. Open the Claude desktop app and start a Cowork session
  3. Point it at ~/claude-inbox and tell it: "When a new .md file appears here, read it and execute the instructions inside it"
  4. Leave Cowork running — it will automatically handle every transcript Plaude drops in

Trigger phrases

Say any of these in your recording. Works for code, writing, planning — anything Claude can do.

send to claude send this to claude claude should claude handle claude fix claude draft claude summarise claude plan claude do this want this to go to claude go to claude for claude claude [anything] →

The catch-all matches any phrase starting with "claude" followed by a word — "claude write me an email", "claude make a plan", "claude research this".

Troubleshooting

cloudflared not found
macOS: brew install cloudflare/cloudflare/cloudflared
Windows: winget install Cloudflare.cloudflared

Then re-run ./start.sh.
Google auth failed
Delete token.json and re-run python setup.py. Make sure you enabled the Google Drive API in Google Cloud Console and that your OAuth credentials are for a Desktop app, not a web app.
Project directory not detected
Plaude scans the transcript for folder names that exist in your projects_root. If the project name isn't mentioned, try saying "working in my-project" or "in the api-service repo".

If nothing matches, a fallback file drops to ~/Desktop/plaude-unmatched/.
claude command not found
Install the CLI: npm install -g @anthropic-ai/claude-code

Verify with which claude. Or switch to the Cowork launcher in config.json to use the desktop app instead.
