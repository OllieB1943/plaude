Setup

Three steps. No installs, no code.

1

Connect Plaud to Google Drive

Open the Plaud app on your phone and enable Google Drive sync. Plaud will automatically upload transcripts to a folder in your Drive after each recording.

1
Open the Plaud app → tap your profile or the settings gear
2
Find Cloud Sync or Backup → tap Google Drive
3
Sign in with your Google account — Plaud creates a folder called PLAUD in your Drive
4
Set sync to Automatic so transcripts upload the moment a recording ends
Tip: Open Google Drive on your computer and find the PLAUD folder. Copy the folder ID from the URL — you'll need it in step 2.
https://drive.google.com/drive/folders/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs
2

Add Google Drive to Claude Cowork

Open the Claude desktop app and start a new Cowork session. Then connect it to the Plaud folder in your Drive.

1
Open Claude desktop app → click Cowork in the sidebar → start a new session
2
Click + Add connector → select Google Drive
3
Sign in with the same Google account your Plaud syncs to
4
Select the PLAUD folder — paste the folder ID from step 1 if asked
3

Give Cowork its instructions

Paste the prompt below as Cowork's first message. It tells Claude to watch the folder, detect trigger phrases, and act on them automatically — forever.

Cowork prompt — paste this once 
Watch my Google Drive Plaud folder for new files. Whenever a new transcript appears:

1. Read the full transcript text
2. Check if it contains a trigger phrase like "send to claude", "claude fix", "claude draft", "claude plan", "claude write", or anything starting with "claude [verb]"
3. If a trigger is found, identify what needs to be done from the transcript and execute it fully — write code, draft documents, do research, send emails, make a plan, or whatever is asked
4. When done, write a brief summary of everything you did

If no trigger phrase is found, ignore the file.
Stay running and repeat this for every new file that appears.
Done. Leave Cowork running in the background. Every time you finish a Plaud recording, the transcript syncs to Drive and Cowork picks it up within seconds — no further action needed.

Trigger phrases

Say any of these in your recording. Works for code, writing, planning — anything Claude can do.

send to claude send this to claude claude fix claude draft claude write claude plan claude research claude summarise claude handle go to claude for claude claude [anything] →

The catch-all matches any phrase starting with "claude" followed by a verb — "claude book me a restaurant", "claude make a plan for this", "claude write a cover letter".

Troubleshooting

Cowork isn't picking up new transcripts
Make sure Plaud's sync is set to automatic in the app settings — not manual. Also verify the Google Drive connector in Cowork is pointing at the correct PLAUD folder. Try making a short test recording and checking if the file appears in Drive within a minute.
I don't see a Google Drive connector in Cowork
Make sure you're on the latest version of the Claude desktop app. The Google Drive connector is available in Cowork sessions via + Add connector. If you don't see it, check for app updates or try restarting the app.
Cowork acted on a recording I didn't intend to send
Cowork only acts when a trigger phrase is detected. Make sure you're not accidentally saying "claude" in recordings you don't want forwarded. You can also edit the prompt in step 3 to use a more specific trigger phrase that's harder to say by accident.
Plaud isn't syncing to Google Drive
In the Plaud app, go to Settings → Cloud Sync → Google Drive and confirm the connection is active. If the folder isn't appearing in Drive, try disconnecting and reconnecting the Google account. Make sure auto-sync is enabled, not just manual backup.
